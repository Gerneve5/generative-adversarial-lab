# Generative Adversarial Lab

![GAN](https://img.shields.io/badge/GAN-Research-blueviolet) ![PyTorch](https://img.shields.io/badge/framework-PyTorch-red)

A laboratory for experimenting with various GAN architectures.

## Architectures
- DCGAN (Deep Convolutional GAN)
- WGAN-GP (Wasserstein GAN with Gradient Penalty)
- CycleGAN for image-to-image translation
